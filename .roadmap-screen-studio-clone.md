## UI Design System

### Colors

- **Accent:** #7C7CFF (purple/blue)
- **Waveform Orange:** #FF6B35
- **Waveform Blue:** #4ECDC4
- **Zoom Purple:** #A855F7
- **Background:** Glassmorphism (.ultraThinMaterial)

### Typography

- Section headers: 10pt, uppercase, semibold, .secondary
- Labels: 11pt, .secondary
- Values: 10pt, monospaced, medium weight

### Spacing

- Inspector spacing: 12px (reduced from 16px)
- Section gaps: 16px
- Control gaps: 8px

### Components

- **SliderRow:** Label + value + Reset button + Slider
- **ToggleRow:** Label + Toggle
- **ColorPickerRow:** Label + ColorPicker
- **PresetGrid:** 3-column grid of thumbnails/swatch
